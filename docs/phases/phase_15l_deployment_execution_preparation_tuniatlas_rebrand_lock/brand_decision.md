# TuniAtlas Brand Decision

## Final public naming

| Item | Value |
|---|---|
| Public brand | TuniAtlas |
| Product line | TuniAtlas Jobs |
| Tagline | Tech careers abroad for Tunisian talent |
| Old public/internal name | TuniTech Abroad, retired publicly |
| App title | TuniAtlas |
| SEO title | TuniAtlas — Tech Jobs Abroad for Tunisians |
| Domain target | tuniatlas.com first, tuniatlas.tn later |
| Repo/project name | Keep `tunitech-abroad` until after deployment |

## Usage rules

Public-facing UI must say **TuniAtlas** or **TuniAtlas Jobs**.

The old name **TuniTech Abroad** may remain only in historical docs, internal phase notes, old repository name, and technical references where renaming would add risk before deployment.

Do not rename Python modules, Django apps, database tables, migrations, Celery queues, or Git remote/repo before deployment.
